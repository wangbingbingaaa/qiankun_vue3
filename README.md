# data-monitor

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve

http://localhost:8088/dataJK/#/  
// 监控系统

http://localhost:8088/dataFX/#/  
// 数据分析系统


```

### Compiles and minifies for production
```
npm run build

npm run dataJK  // 监控系统
npm run dataFX  // 数据分析系统

```
